Manual Test Instructions

Cách test thủ công:
  1. Clear Auth → Truy cập user-list.html → Expect: Redirect to login
  2. Login as User → Truy cập user-list.html → Expect: Access denied + redirect to homepage
  3. Login as Admin → Truy cập user-list.html → Expect: Page loads normally
  4. Login as Admin → Truy cập user-detail.html → Expect: Page loads normally
